
## How to install


```Backend : -

Tech : Nodejs - 14.15.4
       Express
       Sequelize 

API routes :- 
    
Register user   : "/api/users/register", method - POST
Login           : "api/users/login" , method - POST
Update user     : "api/users/" method - PUT
delete user     : "api/users" method - DELETE

To run project move to backend directory and 
run "npm install" & "npm run dev"
for DB : change db details in config/config.json
    run migration "sequelize db:migrate"

Frontend :- 

Tech : react
       Redux
       Axios

to run : move to client directory and run "npm start"

```
